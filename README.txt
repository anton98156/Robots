
Карта с размером  x y.
На ней можно создавать роботов, указывая начальное положение.
Если начальное положение некорректное ИЛИ эта позиция занята другим роботом - программа выбрасывает исключение.
Робот имеет направление (вверх, вправо, вниз, влево). У роботов можно менять направление и передвигать их на 1 шаг вперед
К коду написан контроллер, который выступает посредником между консолью (пользователем) и этой игрой.
Реализовано чтение команд с консоли и их выполнение в "main" методе


(0,0)     -----------------------        (0, y)

(x, 0)    -----------------------        (x, y)


Список команд:
create-map x y (доступна только в начале)
create-robot x y
move-robot id
change-direction id direction

* directions: TOP, RIGHT, BOTTOM, LEFT
* добавлено 20 ходов по умолчанию
